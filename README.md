## ğŸš¦ Getting Started

### Prerequisites
- Node.js 18+
- npm or yarn

### Installation

```bash
# Clone the repository
git clone [your-repo-url]
cd smitdesai

# Install dependencies
npm install

# Run development server
npm run dev
```

### Build for Production

```bash
# Create optimized production build
npm run build

# Start production server
npm start
```
